SadBoat

Who are we?

  We're a small indie games studio operating out of the United States. We develop all of our own software in house. Our flagship software is the SadBoat engine that is used for all of our titles.

  The SadBoat engine is an Open Source, free to use engine built in C++ for use with low-level object oriented game programming. It offers memory efficient 2D functionality with pixel perfect collisions, multithreading, and real-time physics. The engine renders using OpenGL and a few other libraries.

  We have plans to expand the engine into a fully fledged tool, with 3D functionality and more accurate physics simulations. Our current estimated release date for Version 1.0 of the 2D engine is scheduled for Q2 2023.

  Our engine will remain free to use indefinitely; so source code contributions are appreciated! You can view the source code for our engine here.
